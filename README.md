## 🛠️ Run Locally

**Prerequisites:** Node.js

1.  **Clone & Install**:
    ```bash
    git clone https://github.com/kaushik943/prompt-verse.git
    cd prompt-verse
    npm install
    ```

2.  **Start Dev Server**:
    ```bash
    npm run dev
    ```
